Tabletop Playground API

This is the API definition for Tabletop Playground, the modern digital tabletop game for fans and creators! Play solo or with up to eight players online in a physically simulated 3D environment. Create your own games with the in-game editor, and enhance them using JavaScript: https://tabletop-playground.com/knowledge-base/scripting-basics/. You can also find the schema definitions for the object template json files here: https://github.com/plasticity-studios/tabletop-playground-schemas
