For example, with port 9229 (the default), open the following URL to open the DevTools: devtools://devtools/bundled/inspector.html?experiments=true&v8only=true&ws=localhost:9229 For Visual Studio Code, add the following to your debug configurations: { "name": "Inspector", "type": "node", "protocol": "inspector", "request": "attach", "address": "localhost", "port": 9229 }

    Parameters
