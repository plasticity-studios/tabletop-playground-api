setImage

  • setImage(textureName: string, packageId?: string): ImageWidget
    • Defined in index.d.ts:2750

    Set the displayed image file.

    Parameters

    • textureName: string

      The filename of the image to load

    • Optional packageId: string

      The id of the package that contains the image file (in the Textures folder). Can be empty when used from scripts to use the same package that contains the script file (same as passing refPackageId). You can find package ids in the manifest.json file in package folders. Usually you won't use this parameter, unless you have a specific reason to load an image from a different package than where the script is located.

    Returns ImageWidget

setImageSize

  • setImageSize(width: number, height: number): ImageWidget
    • Defined in index.d.ts:2740

    Set the desired size of the image. If both axes are set to 0, the size will be the actual size of the image file. If only one axis is set to 0, it will be set to a value that preserves the aspect ratio of the image, and the other value above 0 will be fixed. Desired size can get overridden by the widget size on a Canvas or the size of the UIElement when the image is the only widget and UIElement.useWidgetSize is false.

    Parameters

    • width: number

      Desired width of the image. Default: 0

    • height: number

      Desired height of the image. Default: 0

    Returns ImageWidget
